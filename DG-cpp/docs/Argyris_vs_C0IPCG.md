# Argyris 元 vs. C⁰ 内罚 Galerkin:求解双调和方程的对比

> 适用程序:`biharmonic_argyris`(本次新增,Argyris C¹ 协调元)与
> `biharmonic_ipcg`(已有,C⁰ 内罚间断 Galerkin)。两者都在
> `Poisson-cpp/` 中,用同一套网格 / 制造解 / CHOLMOD 求解器。

---

## 1. 问题与制造解

在单位正方形 $\Omega=(0,1)^2$ 上求解**固支(clamped)双调和**问题

$$
\Delta^2 u = f \quad\text{in }\Omega,\qquad
u = g_1,\ \ \partial_{\mathbf n} u = g_2 \quad\text{on }\partial\Omega .
$$

制造解 $u(x,y)=\sin\big(\pi(x-c)\big)\sin\big(\pi(y-c)\big)$($c=0.3$),
$f=\Delta^2u=4\pi^4 u$,边界数据 $g_1,g_2$ 取自精确解。

两种方法都基于同一个能量(弯曲能):

$$
a(u,v)=\int_\Omega D^2u : D^2v
      =\int_\Omega \big(u_{xx}v_{xx}+u_{yy}v_{yy}+2u_{xy}v_{xy}\big),
\qquad (f,v)=\int_\Omega f\,v .
$$

差别只在**如何让离散空间逼近 $H^2$**。

---

## 2. 两种方法

### 2.1 Argyris 元(C¹ 协调)

- **局部空间**:五次多项式 $P_5$,每个三角形 21 维。
- **21 个自由度**:
  - 每个顶点 6 个:$u,\ u_x,\ u_y,\ u_{xx},\ u_{xy},\ u_{yy}$(共 18);
  - 每条边中点 1 个:法向导数 $\partial_{\mathbf n} u$(共 3)。
- **全局空间** $\subset H^2(\Omega)$(真正的 $C^1$ 协调),因此**直接用连续弯曲能** $a(u,v)=\int D^2u:D^2v$,**没有任何内罚 / Nitsche 项**。
- **本实现的关键点**(见 `src/argyris/Argyris.{h,cpp}`):
  - 逐单元构造节点基:把 21 个自由度泛函作用在 $P_5$ 重心基上,组装 $21\times21$ 矩阵 $V_t$ 并求逆得 `coef[t]`;顶点的导数自由度用**全局笛卡尔坐标**表示(天然跨单元共享),边法向导数用**全局固定的边法向**(两侧单元得到完全相同的泛函,无需符号处理),从而保证全局 $C^1$。
  - **固支强边界条件的微妙之处**:为使检验函数在边界上 $v=0,\ \partial_{\mathbf n}v=0$,$C^1$ 迹的论证要求在每个边界顶点固定:函数值、整个梯度、$u_{xy}$ 以及**二阶切向导数**;唯一保留**自由**的是**纯二阶法向导数** $\partial_{\mathbf{nn}}u$。正是“恰好让 $\partial_{\mathbf{nn}}u$ 自由”这一点保住了 $L^2$ 的 Aubin–Nitsche 超收敛($k{+}1$ 阶);角点处两个法向相遇,6 个顶点自由度全部固定。
    > 若把边界顶点的 Hessian 也全部钉死到精确值,方法仍然**相容**,$H^2$ 与 $H^1$ 阶不变,但 $L^2$ 阶会从 6 掉到 5(实测验证过)。

### 2.2 C⁰ 内罚 Galerkin(C⁰-IPCG)

- **局部空间**:标准 Lagrange $P_k$($k\ge2$),全局只 $C^0$ 连续(就是普通协调 $P_k$ 元,**不是** $H^2$ 协调)。
- 由于跨单元法向导数不连续,直接用 $\int D^2u:D^2v$ 不相容,需要在内部边上加**内罚项**惩罚 $\partial_{\mathbf n}u$ 的跳量,并补上对称/相容项:

$$
P(u,v)=\sum_{e}\int_e\Big(-\{\!\{\partial_{nn}u\}\!\}[\![\partial_n v]\!]
   -\beta\{\!\{\partial_{nn}v\}\!\}[\![\partial_n u]\!]
   +\tfrac{\sigma}{h_e}[\![\partial_n u]\!][\![\partial_n v]\!]\Big),
$$

  $\beta=1$ 为 SIPG,$-1$ 为 NIPG,$0$ 为 IIPG;**罚参数** $\sigma=\tfrac32 k(k{+}1)$ 需取得足够大以保证强制性。
- 固支边界的 $\partial_{\mathbf n}u=g_2$ 通过 **Nitsche** 弱施加($u=g_1$ 仍强施加);见 `src/ipdg/DGAssembly.cpp` 与 `biharmonic_ipcg_main.cpp`。

## 3. 结构性对比

| 维度 | Argyris(C¹ 协调) | C⁰-IPCG |
|---|---|---|
| 局部多项式 | 固定 $P_5$ | 任意 $P_k,\ k\ge2$ |
| 每单元自由度 | 21(含一二阶导数) | $(k{+}1)(k{+}2)/2$(仅节点值) |
| 全局连续性 | $C^1$($\subset H^2$),真协调 | $C^0$,非协调(靠内罚补偿) |
| 双线性型 | 纯能量 $\int D^2u:D^2v$,**无罚** | 能量 **+ 内罚 + Nitsche** |
| 可调参数 | 无 | 罚参数 $\sigma$、内罚型 $\beta$ |
| 边界条件 | 强施加(需切/法向分解,角点特殊) | $u$ 强施加 + $\partial_n u$ Nitsche 弱施加,简单统一 |
| 网格/几何 | 仿射三角形;曲边、各向异性、自适应较繁琐(非仿射等价) | 标准 Lagrange,曲边/自适应/任意 $k$ 都容易 |
| 实现复杂度 | 高:逐单元 $21\times21$ 节点基、导数自由度共享、边法向定向、边界自由度的切/法向分解 | 中:复用已有 $P_k$ 元 + 边积分装配 |
| 条件数(同阶) | 较好(纯椭圆能量) | 较差($\sim\sigma h^{-4}$,高阶时更糟) |

## 4. 数值结果(P5,固支,SIPG;`ExactSolution(0.3)`,Nref=5)

运行命令:

```bash
./build/biharmonic_argyris                 # Argyris P5
./build/biharmonic_ipcg 5 CLAMPED 5 SIPG   # C0-IPCG: ord=5, 固支, Nref=5, SIPG
```

### 4.1 误差与收敛阶

**Argyris(P5)** —— $H^2$ 半范阶 $\approx k{-}1=4$,$H^1\approx5$,$L^2\approx k{+}1=6$:

| $h$ | nDof | $\|u-u_h\|_{L^2}$ | $\|u-u_h\|_{H^1}$ | $|u-u_h|_{H^2}$ |
|---|---|---|---|---|
| 1/2  | 70   | 1.47e-03 | 1.86e-02 | 2.88e-01 |
| 1/4  | 206  | 1.57e-05 | 4.50e-04 | 1.60e-02 |
| 1/8  | 694  | 1.65e-07 | 1.05e-05 | 8.42e-04 |
| 1/16 | 2534 | 2.00e-09 | 2.74e-07 | 4.75e-05 |
| 1/32 | 9670 | **2.69e-11** | 7.75e-09 | 2.81e-06 |
| 阶 (最细两层) | | **6.22** | **5.14** | **4.08** |

**C⁰-IPCG(P5,SIPG)** —— 良条件区间内 $L^2\approx6,\ H^1\approx5$:

| $h$ | nDof | $\|u-u_h\|_{L^2}$ | $\|u-u_h\|_{H^1}$ |
|---|---|---|---|
| 1/2  | 121   | 2.96e-04 | 4.22e-03 |
| 1/4  | 441   | 4.20e-06 | 1.35e-04 |
| 1/8  | 1681  | 6.08e-08 | 4.20e-06 |
| 1/16 | 6561  | 9.21e-10 | 1.30e-07 |
| 1/32 | 25921 | 2.12e-09 ⚠️ | 1.22e-08 |
| 阶 (1/8→1/16) | | 6.04 | 5.01 |
| 阶 (1/16→1/32) | | **−1.20** ⚠️ | 3.42 ⚠️ |

⚠️ **最细一层 C⁰-IPCG 的 $L^2$ 误差不降反升(阶 $-1.2$)**:P5 + 内罚的全局系统条件数随 $h$ 大致 $\sim h^{-4}$ 且因高阶进一步恶化,在 $h=1/32$($\sim2.6\times10^4$ 未知量)时直接 Cholesky 已被舍入/病态限制在 $\sim10^{-9}$ 量级,精度饱和。Argyris 在同一网格上仍干净地降到 $2.7\times10^{-11}$。

### 4.2 规模与耗时(最细网格 $h=1/32$)

| | Argyris P5 | C⁰-IPCG P5 |
|---|---|---|
| 全局自由度 | **9 670** | 25 921($\approx2.7\times$) |
| 装配耗时 | 0.148 s | 0.249 s |
| 求解耗时(CHOLMOD) | **0.063 s** | 0.468 s($\approx7.4\times$) |
| 最细层 $L^2$ 误差 | **2.7e-11** | 2.1e-9(已饱和) |

同阶精度下,Argyris 自由度更少、矩阵更良态、求解更快;C⁰-IPCG 因每单元更多节点自由度 + 内罚带来的稠密耦合与病态,规模与求解成本都明显更高。

## 5. 优劣总结

### Argyris 元的优点
1. **真 $H^2$ 协调**:无内罚、无 Nitsche、无罚参数需要调,理论与实现都“干净”;Galerkin 正交性直接成立。
2. **同阶自由度更少、系统更良态、求解更快**(见上表)。
3. **直接给出 $C^1$ 解**:梯度、Hessian 处处连续,作为节点自由度可直接取用——对板壳、相场、表面张力等需要 $\nabla u$/曲率连续的问题很有价值。
4. 收敛干净,无罚参数过小导致的不稳定、过大导致的病态之间的权衡。

### Argyris 元的缺点
1. **实现复杂**:21 个含一/二阶导数的自由度、逐单元节点基、边法向定向、**边界条件需做切/法向分解且角点特殊处理**(本文 §2.1)。
2. **多项式次数固定为 5**(经典 Argyris),不像 IPDG 那样换 $k$ 即可;低阶/高阶都不灵活。
3. **非仿射等价**:曲边单元、各向异性、$hp$/自适应、三维推广都要额外的变换处理,工程上更繁琐。
4. 顶点二阶导数自由度对**网格质量更敏感**(局部 $V_t$ 条件数 $\sim h^{-4}$;本实现用 `fullPivLu` 求逆以保稳健)。

### C⁰-IPCG 的优点
1. **实现简单**:直接复用标准 Lagrange $P_k$ 元(本仓库的 `FEM`),只需加边上的内罚 / Nitsche 装配;边界条件统一(强施 $u$ + 弱施 $\partial_n u$)。
2. **任意阶 $k\ge2$**,$hp$、曲边、自适应、复杂几何都自然;易并行装配。
3. 标准 $C^0$ 数据结构,网格生成与可视化无需特殊节点。

### C⁰-IPCG 的缺点
1. **需要罚参数** $\sigma$:太小失稳(失去强制性),太大加剧病态;需要经验/估计。
2. **同阶下自由度更多、条件数更差**:高阶 + 细网格时尤其明显,本实验 P5、$h=1/32$ 即出现 $L^2$ 精度饱和/反弹。
3. **仅 $C^0$**:解的法向导数跨单元不连续,$\nabla u$/曲率不是协调连续的;需要时还得后处理。
4. 非协调带来一致性误差项,需要对称化与精心实现才能拿到最优阶。

## 6. 该用哪个?

- **要 $C^1$ 解 / 协调性 / 无参数 / 高精度且固定四阶能量**(板弯曲、Cahn–Hilliard、几何流等),且网格是较规则的三角形:**Argyris** 更优——更少自由度、更良态、更快、解更光滑。
- **要灵活的阶数、复杂或曲边几何、$hp$-自适应,或希望复用现成 $C^0$ 元、快速落地**:**C⁰-IPCG** 更省事,代价是要调罚参数、系统更大更病态。

一句话:**Argyris 用“复杂的单元”换“干净、良态、低自由度的离散系统”;C⁰-IPCG 用“简单灵活的单元”换“需要调参、规模更大、条件更差的内罚系统”。**

## 7. 复现

```bash
# 在 Poisson-cpp/ 下(Windows + w64devkit,见 README 第 2 节)
cmake -S . -B build -G "MinGW Makefiles" \
      -DCMAKE_TOOLCHAIN_FILE=w64devkit-toolchain.cmake -DCMAKE_BUILD_TYPE=Release
cmake --build build -j

./build/biharmonic_argyris                 # 可选: [Nref] [solver]
./build/biharmonic_ipcg 5 CLAMPED 5 SIPG   # [ord] [bc_type] [Nref] [ip_type]
```

> 说明:为公平对比,两者都取 $P_5$、固支边界、同一网格序列与 CHOLMOD。
> Argyris 的 $H^2$ 半范阶为 $k{-}1=4$,与 C⁰-IPCG($P_5$)的能量阶一致;
> 二者 $L^2$ 理论阶均为 6,$H^1$ 为 5,差别体现在自由度规模、条件数与稳健性上。
